# Lambdas

Una lambda es una función en una línea que sólo realiza una operación, la cual recibe sus debidos argumentos y regresa el valor sin necesidad de un `return`, su sintaxis es la siguiente:

~~~python
my_lambda = lambda ...Lista de argumentos, si no es ninguno lo dejamos en blanco... : ...operación a realizar…
~~~

#### Ejemplo:

~~~python
sum = lambda a,b : a + b
~~~
